# 포트폴리오 및 종목 관리 가이드 (Portfolio & Stock Management Guide)

## 1. 개요
본 문서는 관심 종목을 검색하여 포트폴리오를 구성하고, 자산 현황과 수익률을 관리하는 방법을 안내합니다.

## 2. 종목 관리

### 2.1 종목 검색
- **검색창**: 대시보드 상단 검색바 이용
- **검색어**: 종목명(예: 삼성전자) 또는 종목코드(예: 005930) 입력
- **상세 정보**: 검색 결과 클릭 시 실시간 시세, 차트, 호가 정보 확인 가능

### 2.2 관심 종목 (Watchlist)
- **추가**: 종목 상세 페이지에서 '관심 종목 추가(☆)' 버튼 클릭
- **확인**: 대시보드 '관심 종목' 위젯에서 등락률 실시간 모니터링
- **삭제**: 리스트에서 '삭제(X)' 버튼 또는 상세 페이지에서 별표 해제

## 3. 포트폴리오 관리

### 3.1 포트폴리오 생성
1. **메뉴 이동**: '포트폴리오' 메뉴 > '새 포트폴리오 만들기'
2. **설정**:
   - **이름**: 포트폴리오 식별 이름 (예: 장기 투자용, 단타용)
   - **연동 계좌**: 매매를 수행할 증권사 계좌 선택
   - **초기 자금**: 관리할 현금 규모 설정 (수익률 계산 기준)

### 3.2 포지션(보유 종목) 관리
- **자동 동기화**: 증권사 계좌와 연동된 경우, 실제 보유 종목이 자동으로 불러와집니다.
- **수동 추가**: 연동하지 않은 경우, '포지션 추가' 버튼을 통해 종목, 수량, 매수단가를 직접 입력할 수 있습니다.
- **수정/삭제**: 오기입된 정보는 수정하거나 삭제할 수 있습니다. (실제 계좌 매매에는 영향 없음)

### 3.3 자산 현황 확인
- **대시보드**: 전체 포트폴리오의 총 평가액, 일간 손익, 총 수익률 요약
- **상세 분석**:
  - **자산 배분**: 종목별, 섹터별 비중 파이 차트
  - **수익률 추이**: 기간별(1주, 1개월 등) 자산 변동 꺾은선 그래프
  - **실현 손익**: 매도 완료된 거래의 확정 수익 내역

## 4. 실시간 차트 활용
- **TradingView 차트**: 전문적인 차트 분석 도구 제공
- **지표 추가**: 이동평균선(MA), 볼린저 밴드, RSI 등 보조지표 설정 가능
- **드로잉**: 추세선, 지지/저항선 작도 및 저장
