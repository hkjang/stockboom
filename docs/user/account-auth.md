# 계정 및 인증 가이드 (Account & Authentication Guide)

## 1. 개요
본 문서는 StockBoom 서비스 이용을 위한 회원가입, 로그인, 보안 설정 및 증권사 연동 방법을 안내합니다.

## 2. 계정 생성 및 로그인

### 2.1 회원가입
1. **접속**: 메인 페이지 우측 상단 '회원가입' 버튼 클릭
2. **정보 입력**: 이메일(ID), 비밀번호(영문/숫자/특수문자 포함 8자 이상), 이름 입력
3. **이메일 인증**: 입력한 이메일로 발송된 인증 링크 클릭하여 가입 완료

### 2.2 로그인
- 이메일과 비밀번호를 입력하여 로그인
- **비밀번호 분실 시**: '비밀번호 찾기'를 통해 이메일로 재설정 링크 수신

## 3. 보안 설정 (2단계 인증)

### 3.1 2단계 인증(2FA)이란?
로그인 시 아이디/비밀번호 외에 스마트폰 앱(Google OTP 등)에서 생성된 일회용 코드를 추가로 입력하여 보안을 강화하는 기능입니다.

### 3.2 설정 방법
1. **설정 메뉴 이동**: 로그인 후 우측 상단 프로필 > 설정 > 보안 탭
2. **활성화**: '2단계 인증 활성화' 버튼 클릭
3. **앱 등록**: Google Authenticator 또는 Authy 앱으로 화면의 QR 코드 스캔
4. **코드 확인**: 앱에 표시된 6자리 인증 코드 입력
5. **완료**: 설정 완료 메시지 확인 (백업 코드는 안전한 곳에 보관)

## 4. 증권사 API 연동

### 4.1 준비 사항
- 한국투자증권 계좌 보유
- 한국투자증권 홈페이지에서 Open API 신청 및 App Key/Secret 발급

### 4.2 연동 절차
1. **계좌 관리 이동**: 대시보드 > 설정 > 증권사 계좌
2. **계좌 추가**: '계좌 연결하기' 버튼 클릭
3. **정보 입력**:
   - 증권사 선택 (한국투자증권)
   - 계좌번호 (하이픈 제외)
   - App Key (발급받은 키 복사/붙여넣기)
   - App Secret (발급받은 시크릿 복사/붙여넣기)
   - **모의투자 여부**: 실전 투자가 아닌 경우 반드시 체크
4. **연동 확인**: '연동 테스트' 버튼을 눌러 정상 연결 확인 후 저장

### 4.3 개인정보 보호
- 입력하신 API Key와 Secret은 암호화되어 안전하게 저장됩니다.
- StockBoom은 사용자의 동의 없이 임의로 매매를 수행하지 않습니다.
