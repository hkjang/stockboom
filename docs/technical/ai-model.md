# AI 모델 및 분석 가이드 (AI Model & Analysis Guide)

## 1. 개요
본 문서는 StockBoom 시스템에서 활용하는 AI 분석 모델의 구조, 학습 주기, 신호 발생 기준 및 이상치 처리 방법을 설명합니다.

## 2. 분석 모델 구조

### 2.1 모델 파이프라인
1. **데이터 전처리 (Preprocessing)**
   - 결측치 보간 (Interpolation)
   - 정규화 (Normalization): Min-Max Scaling
   - 피처 엔지니어링: 이동평균, RSI, MACD 등 기술적 지표 추가

2. **모델 구성**
   - **시계열 예측**: LSTM (Long Short-Term Memory) 또는 Transformer 기반 모델
   - **패턴 인식**: CNN (Convolutional Neural Network) 기반 차트 이미지 분석
   - **감성 분석**: BERT 기반 뉴스 텍스트 분류 모델 (긍정/부정/중립)

3. **앙상블 (Ensemble)**
   - 각 모델의 예측 결과를 가중 평균하여 최종 점수 산출

## 3. 학습 및 업데이트 주기

### 3.1 정기 학습 (Retraining)
- **주기**: 매주 주말 (시장 휴장 시간 활용)
- **데이터**: 최근 1년치 시세 및 뉴스 데이터
- **검증**: 최근 1개월 데이터를 Test Set으로 사용하여 성능 평가 (Accuracy > 60% 유지 목표)

### 3.2 온라인 학습 (Online Learning) - *고급*
- 매일 장 마감 후 당일 데이터를 반영하여 모델 가중치 미세 조정 (Fine-tuning)

## 4. 신호 발생 및 점수 산출

### 4.1 AI 점수 (AI Score) 산출 방식
- **범위**: 0 ~ 100점
- **구성 요소**:
  - 기술적 추세 (Trend): 40%
  - 패턴 일치도 (Pattern): 30%
  - 뉴스 감성 (Sentiment): 20%
  - 수급 강도 (Volume): 10%

### 4.2 매매 신호 기준
| 신호 | 점수 구간 | 의미 | 권장 행동 |
|------|-----------|------|-----------|
| **Strong Buy** | 80 ~ 100 | 강력 매수 | 적극 매수 진입 |
| **Buy** | 60 ~ 79 | 매수 | 분할 매수 진입 |
| **Hold** | 40 ~ 59 | 관망 | 기존 포지션 유지 |
| **Sell** | 20 ~ 39 | 매도 | 일부 수익 실현 |
| **Strong Sell** | 0 ~ 19 | 강력 매도 | 전량 매도 또는 손절 |

## 5. 이상치 및 노이즈 처리

### 5.1 이상치 탐지 (Anomaly Detection)
- **가격 급변**: 전일 대비 ±30% 이상 변동 시 모델 입력에서 제외하거나 별도 플래그 처리
- **데이터 오류**: 거래량이 0이거나 가격이 음수인 경우 전처리 단계에서 필터링

### 5.2 노이즈 제거
- **스무딩 (Smoothing)**: 이동평균 등을 사용하여 일시적인 가격 튀는 현상 완화
- **뉴스 필터링**: 광고성 기사나 중복 기사는 감성 분석 대상에서 제외
